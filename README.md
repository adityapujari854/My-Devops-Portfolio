# Aditya Pujari — DevOps Portfolio

This repository is a Next.js + TypeScript + Tailwind CSS portfolio scaffold built for a DevOps / SRE focused resume site.

Getting started:

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm start
```

Notes:
- This scaffold includes components, data files, and a GitHub integration stub. Replace placeholder assets in `public/` and update `data/` with your real content.
- Deploy to Vercel for best results.
