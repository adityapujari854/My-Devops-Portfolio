# 🚀 DevOps Portfolio Website

A modern, responsive, and production-inspired personal portfolio built to showcase my journey as an **AWS DevOps & Cloud Engineer**. The website highlights cloud infrastructure, automation, containerization, CI/CD pipelines, AI-assisted engineering, and real-world DevOps projects through a clean, professional interface.

## ✨ Features

* Modern responsive UI with dark/light mode
* Built using **Next.js 15** and **TypeScript**
* Smooth animations powered by **Framer Motion**
* Premium glassmorphism-inspired design
* Interactive DevOps terminal animation
* GitHub API integration for repositories and statistics
* Resume download
* Contact form
* SEO optimized
* Fully responsive across desktop, tablet, and mobile devices
